Боты в репозитории

1. Музыкальный бот
   Бот для поиска музыки.

Функционал: поиск треков по названию, генерация прямых ссылок.

2. Бот-счетчик (Поисковик)
   Учебный бот для тренировки работы с переменными и базами данных.

Функционал:

Создает ссылку для поиска в Google.

Считает количество запросов пользователя.

Имеет интерактивные кнопки.

Технологии
Python 3.x

Aiogram 3.x

Как запустить бота
Клонируйте репозиторий:
git clone https://github.com/ВАШ_НИК/my-first-bots.git

Установите библиотеки:
pip install aiogram

Добавьте токен:
Откройте файл бота и вставьте свой токен от BotFather в переменную API_TOKEN.
